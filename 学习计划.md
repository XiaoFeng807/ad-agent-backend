## 二、学习优先级（按顺序）

### 第一优先：Git（今天可学会）

只需掌握 4 句命令：

```bash
git init                    # 把当前文件夹变成 Git 仓库
git add .                   # 把所有改动添加到暂存区
git commit -m "做了什么"    # 提交，生成一个版本
git log                     # 查看提交历史
```

验收标准：在你的项目里 `git init`，然后 `git add .`，`git commit -m "初始化项目"`，`git log` 能看到记录。

### 第二优先：看完项目核心文件

按顺序看，每看完一个就尝试用自己的话讲出来：

```
1. backend/agent/agent.py     ← 已看完 ✅
2. backend/tools/tools.py     ← 24 个函数，挑 3 个最感兴趣的逐行理解
3. backend/auth/auth.py       ← JWT Token 怎么生成和验证
4. backend/database/database.py ← 数据库表结构、初始化、种子数据
5. backend/routes/alert_routes.py ← 刚修过的，前后端怎么通信
```

验收标准：每个文件我能问你 3 个问题，你能答上来。

### 第三优先：SQL

你项目里的 SQL 全是这 4 种：

```sql
-- 查询
SELECT * FROM users WHERE id = 1

-- 插入
INSERT INTO users (username, password, role) VALUES (?, ?, ?)

-- 更新
UPDATE users SET role = ? WHERE id = ?

-- 删除
DELETE FROM users WHERE id = ?
```

验收标准：打开 database.py，能看懂里面每条 SQL 在干什么。

### 第四优先：手写 3 个 Function Calling 函数

选一个免费 API（新闻、股票、笑话都可以），按这个模板写：

```python
def get_xxx(param):
    """函数描述"""
    url = "https://api.xxx.com/endpoint"
    headers = {"User-Agent": "Mozilla/5.0"}
    resp = requests.get(url, headers=headers)
    data = resp.json()
    # 处理数据
    return {"result": data}
```

验收标准：函数能跑通，返回你想要的数据。然后把它注册到项目的 tools.py 里让 AI 能调用。
